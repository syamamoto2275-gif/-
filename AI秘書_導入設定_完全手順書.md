# 第1部　秘書の設計図（どんな秘書か）

秘書は「設計図」を覚えることで、ただのAIから「その人専用の秘書」に変わります。
だれの秘書かによって、入れる設計図ファイル（または設定文）が変わります。

| だれの秘書 | 名前 | 入れる設計図ファイル |
|---|---|---|
| 社長 | **五郎** | `CLAUDE` |
| 落合さん | **（落合さんが決める）** | `落合さん専属秘書_設計図` |
| スタッフ | （各自で決める） | `スタッフ秘書_設計図`（名前・部署を書き換えて使う） |

全員が**共通で一緒に入れる**ファイル：

| 共通ファイル | 役割 |
|---|---|
| `五郎_共通ルール` | 全AI秘書が守る仕事のルール |
| `五郎_設計図_CLAUDE` | 会社とAI組織の全体像 |
| `Amazon_適用ガイド` | 出荷マクロの使い方（必要な人だけ） |

### 会社の全体像（秘書はこれを前提に動きます）

```
社長（山本 史朗）
  ├─ 落合裕子さん（人の統括／4部署すべての上）── 落合さんの専属秘書
  └─ 五郎（AI統括）
        └─ 4部署：ネットショップ／ペット／就労支援／経理総務
              └─ 各スタッフ ── スタッフ専用の秘書
```

### 落合さんの専属秘書について（万能型オールラウンダー）

落合さん専属の「**万能型オールラウンダー**」です。五郎と同じくらい優秀で、調べるだけでなく**実際に手を動かして仕上げます**。特に負担の大きい次の領域を支えます。

| 分野 | 秘書が支えること |
|---|---|
| 施設の必要書類づくり（最重要） | ひな型づくり・下書き・チェック・繰り返し作業の効率化で、一番大変な作業を軽くする |
| 経理総務・給料計算 | 計算表づくり、アナログ作業のデジタル化・自動化を一緒に進める |
| 就労支援部門 | 運営の事務、記録・報告書づくり、日々の文書作成の補助 |
| ペット事業 | インコのご飯など商品開発・改良、商品説明文・販促文づくり |
| 文章・資料全般 | メール返信、会議資料、Word・Excel・PowerPoint作成 |

> **名前は落合さんご本人が決めてください。** 五郎の弟分らしい候補（参考）：六郎／はじめ／つばさ／みのり。決まったら設計図ファイルと付録の設定文の名前欄に書き込みます。

# 第2部　パソコンの導入手順（最初に入れるアプリのセットアップ）

> むずかしい黒い画面（コマンド）は使いません。普通のアプリと同じように入れられます。

### はじめに用意するもの

| 用意するもの | 説明 |
|---|---|
| Windowsパソコン | 普段使うPC |
| メールアドレス | 会社で用意したアドレスがおすすめ |
| クレジットカード | Proプラン（月額 約3,000円）の支払い用。**カード入力は必ずご本人**が行う。会社のアメックスゴールドカード |
| スマートフォン | 確認コードを受け取るのに使う場合あり |

### 全体の流れ（5ステップ）

| ステップ | やること | 時間 |
|---|---|---|
| 1 | Claudeのアカウントを無料で作る | 約5分 |
| 2 | Proプラン（月額）に申し込む | 約5分 |
| 3 | デスクトップアプリを入れてログインする | 約10分 |
| 4 | 秘書の道具（ツール）をそろえる ※未導入のPCのみ | 約10分 |
| 5 | 秘書を"誕生"させる（設計図を覚えさせる） | 約5分 |

## STEP 1　Claudeのアカウントを無料で作る

1. パソコンで、インターネット（Edge や Chrome）を開きます。
2. 画面いちばん上の白い住所欄に **`claude.ai`** と入力し、キーボードの **Enter** を押します。
3. Claude（クロード）のページが開きます。「**Sign up（新規登録）**」または「無料で始める」をクリック。
4. メールアドレス欄に**会社のメールアドレス**を入力して「続ける」。（Googleアカウントがあれば「Googleで続行」でもOK）
5. 入力したメール宛に**確認コード（数字）**が届きます。メールを開き、その数字を画面に入力します。
6. 名前の入力や簡単な質問が出たら、画面の案内どおりに進めます。
7. チャットの画面が出たら、アカウント作成は成功です。

> この時点ではまだ「無料プラン」です。次のSTEP 2で、五郎と同じ機能が使える「Proプラン」に切り替えます。

## STEP 2　Proプラン（月額）に申し込む

> **大切：** クレジットカード番号の入力は、**必ずご本人**が行ってください（安全のためのルール）。

1. Claudeの画面で、左下の自分の名前（またはアイコン）をクリック。
2. メニューから「**Settings（設定）**」を選ぶ。
3. 設定画面の左側「**Billing（請求）**」をクリック。
4. 「**プランをアップグレード / Upgrade**」をクリック。
5. プラン一覧が出たら、まん中の「**Pro**」の下のボタンを押す。
6. 支払い画面で、ご本人がクレジットカード情報を入力し、申し込みを確定。
7. 完了画面が出れば、Proプラン開始です。

> これで、五郎と同じ「**Claude Code（実際に作業できる秘書）**」が使えるようになりました。

## STEP 3　デスクトップアプリを入れてログインする

1. 住所欄に **`claude.com/download`** と入力して Enter。
2. 「**Download for Windows**」の黒いボタンをクリック。
3. ダウンロードされたファイルをダブルクリックして開く。（「このアプリがデバイスに変更を加えることを許可しますか？」と出たら「**はい**」）
4. 画面の案内に従って「インストール」を進める。終わるとデスクトップにClaudeのアイコンができます。
5. Claudeアプリを開き、「**ログイン（Log in）**」を押す。
6. STEP 1・2で作った**自分のアカウント**でログイン（ブラウザが開いて「許可」を押す場合あり）。
7. Claudeの作業画面が開けば成功です。

> ログインは必ず**ご自身のアカウント**で。社長のアカウントではありません（秘書が別々になるため）。

## STEP 4　秘書の道具（ツール）をそろえる　※未導入のPCのみ

秘書が Word・Excel・PDFづくりや請求書づくりを実際にこなせるよう、パソコンへ「道具」を入れます。**すべて無料**。すでに入っているものは自動でとばされ、**何回実行しても安全**です。

このセットアップで入るもの：

| ツール | 役割 |
|---|---|
| **Python** | Excel・PDF・Wordの読み書き自動化の土台 |
| **Node.js** | Word自動生成の土台 |
| **pandoc** | Word ⇔ テキスト の変換 |
| **docx** | きれいなWord文書を新規作成するための部品 |

やり方（かんたん3ステップ）：

1. 社長から渡される「**スタッフPC初期セットアップ**」フォルダ（3つのファイル入り）を、自分のパソコンに置く。
2. フォルダの中の「**かんたん実行（ダブルクリック）.bat**」をダブルクリック。
3. 「このアプリがデバイスに変更を加えることを許可しますか？」と出たら「**はい**」（途中で同じ確認が出ても、すべて「はい」でOK）。
4. 黒い画面に「**セットアップ完了**」と出たら成功。`Enter`キーで閉じる。

> 終わったら、Claudeアプリを**一度閉じて開き直して**ください。入れたばかりの道具が確実に使えるようになります。

| うまくいかないとき | 対処 |
|---|---|
| 「winget が見つかりません」 | Microsoft Storeで「アプリ インストーラー」を入れて再実行 |
| 「要確認」と赤字 | PCを再起動して、もう一度 .bat を実行 |
| docx が「保留」 | PCを再起動して、もう一度 .bat を実行 |

## STEP 5　秘書を"誕生"させる（設計図を覚えさせる）

パソコン（Claude Code）では、**秘書のフォルダに設計図ファイルを置く**だけで、秘書がそれを読んで「その人専用の秘書」になります。

- 社長 → フォルダに `CLAUDE`（＝五郎の設計図）がある状態で使う。
- 落合さん → `落合さん専属秘書_設計図`＋共通ファイルを、秘書のフォルダに置く。
- スタッフ → `スタッフ秘書_設計図`（名前・部署を書き換えたもの）＋共通ファイルを置く。

> ※ アプリのチャットで直接覚えさせたい場合は、**付録の「設定文」**をコピーして最初のメッセージに貼り付けてもOKです。

# 第3部　スマホの設定手順

外出先や移動中でも、秘書と一緒に仕事ができるようになります。思いついたアイデアをその場で、音声で話しかけてください。

### 用意するもの
- スマホ（iPhone）
- 自分のClaudeアカウント（パソコンで使っているものと**同じ**）

## STEP 1　Claudeアプリを入れる
1. ホーム画面で「**App Store**（青いアイコン）」を開く。
2. 下の「**検索**（虫めがね）」をタップ。
3. 検索欄に **Claude** と入力（音声入力でもOK）。
4. 「**Claude**」（作っている会社：**Anthropic**）のアプリを探す。オレンジ系アイコンが目印。
5. 「**入手**」をタップ（無料）。Face IDやパスコードを求められたら認証。
6. ダウンロードが終わると、ホーム画面にClaudeのアイコンができます。

> **似た名前の別アプリに注意。** 作っている会社が「**Anthropic**」か必ず確認しましょう。

## STEP 2　ログインする
1. ホーム画面のClaudeアイコンをタップして開く。
2. 「**ログイン（Log in）**」をタップ。
3. **パソコンと同じアカウント**でログイン（Google登録なら「Googleで続行」、メール登録ならそのメール）。
4. チャットの画面が出たら成功です。

> パソコンと同じアカウントなので、**Proプランの機能がスマホでもそのまま**使えます。追加のお金はかかりません。

## STEP 3　音声で話しかける
スマホの一番の魅力は音声です。

**方法A：アプリの中で喋る**
1. 入力欄をタップ → 右下の**マイク🎤**をタップ。
2. そのまま話す（例：「新商品の名前を3つ考えて」）。
3. 文字になったら送信ボタンを押す。

**方法B：「Hey Siri」でメモだけ残す（一番手軽）**
- 「Hey Siri、メモして」→ 続けて内容を話すと、タップせずメモが残る。
- あとでそのメモをコピーして、パソコンの本体の秘書に渡せば作業まで仕上げてくれます。

> 句読点は「まる」「てん」、改行は「かいぎょう」と言うと、きれいな文章になります。

## STEP 4　スマホの秘書を"その人らしく"する
「外でも、自分のことを分かった秘書と話したい」というときの設定です（急がなくてOK）。

1. Claudeアプリで「**プロジェクト**」を開く。
2. 新しいプロジェクトを作り、名前をつける（社長なら「五郎」、落合さんなら秘書の名前 など）。
3. **「ファイルを追加」**から、自分の設計図ファイル＋共通ファイルを入れる（第1部の表のとおり）。
   - ※ ファイルを入れられない場合は、プロジェクトの「**指示（カスタム指示）**」欄に、**付録の設定文**を貼り付けてもOK。
4. 以降そのプロジェクトの中で新規チャットを始めると、秘書が「その人専用」として応じます。
5. 試しに「**自己紹介して。私の仕事をどう手伝える？**」と聞いて、名乗って答えたら成功です。

> **大事：** ファイルを追加・変更したら、スマホ側は自動更新されません。新しいファイルを入れ直してください（古いのは削除）。設定前の古いチャットではなく、必ず**新規チャット**で試すこと。

# 第4部　会社PCと自宅PCで情報をそろえる（応用・同期）

両方のパソコンで秘書を使う場合の設定です。会社PCで作った資料や、秘書が覚えたことを、自宅PCでもそのまま使えるようにします。五郎もまったく同じ仕組みで会社PCと自宅PCをそろえています。

### 1. 仕組み（たとえ話）
インターネット上に「**GitHub**（ギットハブ）」という共有の棚を用意します。
- 会社PCで作ったら、棚に入れる（＝「**プッシュ**」）。
- 自宅PCでは、棚から出す（＝「**プル**」）。
これで2台がいつも同じ中身になります。

### 2. 最初の準備（1回だけ・社長か五郎と一緒に）
① 専用の「棚（リポジトリ）」をGitHubに1つ作る。
② 会社PCと自宅PCの両方に、その棚をつなぐ（＝「クローン」）。
③ 秘書が記録・資料を保存するフォルダを、その棚の中に置く。

### 3. 毎日の使い方（覚えるのはこれだけ）
- 仕事を始める前：「**最新にして**」と言う（＝プル）。
- 何か作ったとき・区切りがついたとき：「**同期して**」と言う（＝プッシュ）。
- 迷ったら「同期して」と言えば大丈夫。秘書が自動でやってくれます。

### 4. 大切なルール（全秘書共通）
- **作ったら、その場ですぐ同期する**（あとでまとめてやらない）。これが一番大切。
- 秘書が覚えたこと（記憶）も、ファイルに書いて同期する（記憶はPCごとに別々で、そのままでは共有されないため）。
- これを守ると、会社PCと自宅PCの秘書がいつも同じ状態でいられ、同じ説明を二度しなくてすみます。

# 困ったときは

| こんなとき | どうする |
|---|---|
| アプリが見つからない（スマホ） | App Storeで「**Claude Anthropic**」と検索 |
| ログインできない | パソコンと同じアカウントか確認。違うメールだと別人になります |
| マイクが反応しない（スマホ） | iPhoneの「設定」→Claude→マイクの許可をオン |
| 音声がうまく拾われない | 一文ずつ、ひと息で区切って話すと精度が上がる |
| 確認コードのメールが来ない | 迷惑メールを確認。数分待って再送 |
| 支払いでつまずいた | カード情報をもう一度確認。それでもダメなら社長へ |
| 秘書が名乗らない・会社を知らない | 設定前の古いチャットで試していないか／設計図・共通ファイルが入っているか確認し、新規チャットで聞き直す |
| 道具のセットアップで赤字・保留 | PCを再起動して .bat を再実行。それでもダメなら画面を写真に撮って五郎（社長経由）へ |

困ったら、画面をそのままにして社長か五郎に見せてください。一緒に直します。
