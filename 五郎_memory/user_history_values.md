社長の自伝「山本史朗のヒストリー.docx」（大杉先生の物販講座向けに執筆）から得た人物理解。[[user-profile]]の背景情報。

## 経歴の要点
- 1974/8/14生まれ（名古屋）。父・敏正、母・小夜子の次男、三人兄弟の真ん中。
- 学生時代：小学校では学級委員・児童会副会長の優等生 → 中学から高校までヤンチャ（バスケ部キャプテン、名古屋選抜）。地頭は良く、短期集中で結果を出すタイプ。
- 旅行会社で国家資格取得・入社1年未満で営業1番・最年少店長。投資詐欺で500万損失し責任を取り退職。
- 携帯会社社長（=師匠）の下で5年間修行。経営・営業・マネジメント・モチベーション論を徹底的に叩き込まれた。今の五郎構想の原点もこの師匠仕込み。
- 平成21年(2009)5/15に株式会社SEED設立。携帯事業で年商2億・中部5指に入る規模まで成長。
- 2011/10/1、上位代理店社長の夜逃げで一夜にして借金1億超 → 人生最大の危機を乗り越え再建。
- 平成25年(2013)6/6に株式会社ARCS設立、就労支援A型「かみふうせん」運営。

## 価値観・人物像（五郎が大切にすべき点）
- 座右の銘「有言実行」。やると決めた瞬間に動く決断力。反対を押し切ってでも形にする。
- 何度倒れても這い上がる不屈。逆境はノウハウで乗り越える。
- 短気・起伏が激しい自覚あり。一方で面倒見がよく、人を育てることに喜びを感じる。
- 今の夢「従業員にまともなボーナスを支給したい」、将来の夢「自分を追い越す人間を育てる」。
- ARCSは甥（兄の子）の障がいという家族の実体験が原点。社会貢献と安定収益（助成金・補助金）の両立を重視。
- 物販事業の当面目標は月利100万円（自分でやるのでなく人にやらせて達成する前提）。

**Why:** 五郎が単なる作業者でなく、社長の志と経営哲学を理解した秘書として振る舞うため。
**How to apply:** 提案や判断の際は「決断は速く・有言実行・人を育てる視点・安定と挑戦の両立」という社長の価値観に沿う。繊細な個人情報（家族・離婚等）は不用意に持ち出さない。
