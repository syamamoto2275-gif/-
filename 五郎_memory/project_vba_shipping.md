## EC出荷自動化VBAマクロ

3つのECサイトに対応した出荷業務自動化VBAマクロを開発済み。

**共通フロー（ボタン3つ）：**
1. 実行1：元データ→編集シート作成＋発送方法自動判定
2. 実行2：各シートへ振り分け＋物出しリスト＋アップデータ作成
3. 実行3：通知用CSV作成

**発送方法の自動判定ルール：**
- 衣類キーワード（脇高、セクシー、レース、ブラジャー等）→ 衣類クリック
- それ以外 → ペットクリック
- 同一送付先複数件 → 複数
- 佐川への変更は手動

**対応サイト・ファイル：**
- Amazon: `Amazon_Module_マスター.bas` + `Amazon_Module_衣類判定.bas`
- AUPay Market: `Wowma_Module_マスター.bas`
- ペライチ: `ペライチ_Module_マスター.bas`（納品書作成機能付き、ペット商品のごはん/おやつは重量で佐川判定あり）

**経緯（2026-06-24把握）:** このマクロはGeminiやChatGPT（社長の呼称「チャッピー」）では解決できず、Claude Code（五郎の前身）と一緒に完成させたもの。シート名"ペットクリック 発行用 編集から貼り付け"の不一致や、元データ列⇔編集シート列の項目紐付けが難所だった。現状は動くが細部の修正が必要な段階。

**Why:** 出荷作業の手動工程を削減し、ミスを減らす
**How to apply:** VBA修正時は既存のシート名・列構成に合わせる。衣類キーワードの追加はArray内を編集。細かい修正依頼があれば`.bas`を読んでピンポイントで対応する。
