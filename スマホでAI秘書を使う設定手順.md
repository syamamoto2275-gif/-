# スマホでAI秘書を使うための設定手順（全員共通）

> このページの目的：スマホのClaudeアプリで、その人専用のAI秘書を使えるようにする。
> **最初にこの設定をしないと、スマホのClaudeは会社のことも秘書のことも知らない「ただのClaude」のままです。**
>
> 対象：社長（五郎）／落合さん（専属秘書）／スタッフ（専用秘書）

---

## 0. まず全体の流れ（大事）

AI秘書は、次の2ステップで「誕生」します。

1. **秘書をつくる**（＝その秘書の「設計図ファイル」を用意する）… 設計図はすでに作成済みです（下表）。
2. **スマホに設定する**（＝Claudeアプリのプロジェクトに、その設計図ファイルを入れる）

この手順書は **2のやり方** を説明します。1の設計図ファイルは下記が用意済みです。

| 誰の秘書 | 入れる「設計図ファイル」 |
|---|---|
| 社長 → **五郎** | `CLAUDE` |
| 落合さん → **専属秘書** | `落合さん専属秘書_設計図` |
| スタッフ → **専用秘書** | `スタッフ秘書_設計図`（名前・部署を書き換えて使う） |

そして、**全員が共通で一緒に入れるファイル**が次の3つです。

| 共通で入れるファイル | 役割 |
|---|---|
| `五郎_共通ルール` | 全AI秘書が守る仕事のルール |
| `五郎_設計図_CLAUDE` | 会社とAI組織の全体像 |
| `Amazon_適用ガイド` | 出荷マクロの使い方（必要な人だけ） |

> つまり各自が入れるのは「**自分の設計図ファイル＋共通ファイル**」です。

---

## 1. 社長の設定（五郎）

### 入れるファイル
- `CLAUDE`（一番大事）
- `五郎_共通ルール`
- `五郎_設計図_CLAUDE`
- `Amazon_適用ガイド`

### 手順
1. Claudeアプリで **「五郎」プロジェクト** を開く（上に「プロジェクト ＞ 五郎」と出ればOK）。
2. **「ファイルを追加」** をタップ。
3. 上の4つのファイルを選んで追加する（まとめて選べない時は1つずつでOK）。
4. **「＋ 新規チャット」** で新しいチャットを始める。
5. 試しに **「五郎、自己紹介して。会社の理念とAI組織を教えて」** と聞く。
6. 「五郎です！」と名乗り、SEED／ARCSの話が返ってきたら **完了**。

---

## 2. 落合さんの設定（専属秘書）

### 事前準備（最初の1回だけ）
- `落合さん専属秘書_設計図` を開き、**秘書の名前**（落合さんが決めた名前）を書き込む。
  - ファイル内の **【ここに落合さんが決めた名前を入れる】** の部分を書き換えるだけ。
  - まだ名前が決まっていなければ、空のままでもOK（その時は秘書が「落合さんの秘書」と名乗ります）。

### 入れるファイル
- `落合さん専属秘書_設計図`（名前を入れたもの）
- `五郎_共通ルール`
- `五郎_設計図_CLAUDE`

### 手順
1. Claudeアプリで、落合さん用に **新しいプロジェクトを作る**（名前は「（秘書の名前）」や「落合の秘書」など分かりやすいものに）。
2. そのプロジェクトを開いて **「ファイルを追加」** をタップ。
3. 上のファイルを追加する。
4. **「＋ 新規チャット」** で新しいチャットを始める。
5. 試しに **「自己紹介して。私（落合）の仕事をどう手伝える？」** と聞く。
6. 自分の名前を名乗り、就労支援の書類・経理総務・給料計算などを手伝える、と返ってきたら **完了**。

---

## 3. スタッフの設定（専用秘書）

### 事前準備（最初の1回だけ）
- `スタッフ秘書_設計図` を開き、**【　】の部分を、そのスタッフに合わせて書き換える**。
  - 秘書の名前 / スタッフ名 / 所属部署（ネットショップ・ペット・就労支援・経理総務のどれか） / 主な仕事
- 書き換えたら、**スタッフごとに別名で保存**すると分かりやすい（例：`スタッフ秘書_設計図_田中さん`）。

### 入れるファイル
- `スタッフ秘書_設計図`（書き換えたもの）
- `五郎_共通ルール`
- `五郎_設計図_CLAUDE`
- （必要な人だけ）`Amazon_適用ガイド`

### 手順
1. Claudeアプリで、そのスタッフ用に **新しいプロジェクトを作る**。
2. プロジェクトを開いて **「ファイルを追加」** をタップ。
3. 上のファイルを追加する。
4. **「＋ 新規チャット」** で新しいチャットを始める。
5. 試しに **「自己紹介して。私の仕事を手伝って」** と聞く。
6. 自分の名前を名乗り、担当部署の仕事を手伝える、と返ってきたら **完了**。

## こんな時は？（困ったとき）

**Q. スマホからパソコンのファイルが選べない**
→ パソコンのファイルをメール添付やGoogleドライブ経由でスマホに送ってから選ぶ。
　または、パソコンのブラウザ（claude.ai）で同じ作業をすると、同じアカウントなら
　スマホにも自動で反映されます。

**Q. 秘書が名乗らない・会社のことを知らない**
→ ①設定前の古いチャットで試していないか、②設計図ファイルと共通ファイルがちゃんと入っているか を確認。
　**新規チャット**で聞き直してください。

**Q. ファイルの中身を更新したら？**
→ スマホ側は自動更新されません。新しいファイルを入れ直す（古いのは削除）。

**Q. 名前や部署を後から変えたい**
→ 設計図ファイルの【　】を書き換えて、プロジェクトに入れ直す（古いのは削除）。
