## 覚えたコマンド

| コマンド | 意味 |
|---|---|
| `cd フォルダ名` | フォルダを移動する |
| `git init` | Git リポジトリを作る |
| `git add ファイル名` | 変更を準備する |
| `git commit -m "メッセージ"` | 変更を記録する |
| `git checkout -b ブランチ名` | 新しいブランチを作る |
| `gh repo create` | GitHub にリポジトリを作る |
| `gh pr create` | PR を作る |
| `gh auth status` | GitHub の接続状態を確認する |

## わかったこと

- **README** = プロジェクトの説明書（一番最初に表示される）
- **ブランチ** = 安全に作業するためのコピー
- **PR（プルリクエスト）** = 「この変更を取り込んでください」という申請
- **マージ** = PRを本番に反映すること
